Teste Técnico Inatel

Uma empresa XYZ deseja criar um sistema para uso interno de seus funcionários, eles necessitam de um sistema que seja seguro e com algumas funcionalidades e você será o responsável por esse desenvolvimento. Funcionalidades necessárias

1. Criar uma interface gráfica para Login e Tela principal do sistema Feito.

2. Tela de Login contendo os campos “usuário” e “senha” previamente cadastrados Feito. Usuário admin criado na inicialização do Docker.

3. Criar um botão que deverá consumir uma API pública (de sua escolha) e exibir os resultados na tela Feito. API de números de Covid 19 Brasil

4. Criar um campo na tela para selecionar um arquivo dos tipos “.xlsx” ou “csv” Feito.
a. Ao clicar em um botão, o arquivo deve:
i. Ser lido, interpretado e seus dados devem ser salvos na base de dados de uma única vez Feito. O arquivo é validado e importado.
ii. O arquivo também deve ser salvo na base de dados Feito. Salvo em media/imports e caminho no banco.
b. Esse arquivo deve ser anexado junto ao projeto para testes Feito.
c. Criar ao menos dois “models” com relacionamento para salvar os dados dos arquivos Feito. Relacionamento da Unidade de Saúde com o Arquivo.

5. Criar uma “view” com “serializers” para exibir dados dos arquivos em uma tela específica Feito. Listar importações -> Clicar em JSON.

6. Criar o Admin Panel para validação dos dados Feito. Admin disponível em /admin.

7. Criar Logs de histórico de tudo que for realizado no sistema. Não feito. Criado apenas o banco separado.
a. Esses Logs deverão ser salvos em uma base de dados diferente da base utilizada pelo ORM default do Django

Requisitos Técnicos

1. Backend: Utilizar Python/Django Feito.

2. Frontend: Escolher a tecnologia de sua preferência Bootstrap.

3. Banco de Dados: PostgreSQL Feito.

a. Os bancos de dados devem rodar no Docker e as instruções de criação e execução devem ser enviadas junto com o projeto Feito. Banco default e logs.

4. Criar teste unitário Feito

Entrega

1. Documentação Feito.

2. Link do repositório onde o projeto estará disponível para download Feito. https://github.com/fassis/inatel

3. Instruções de cópia, instalação e execução do projeto em qualquer máquina em ambiente local Feito.
